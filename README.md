# Projeto Elo Fácil
Este é um projeto de website front-end estático para a "Elo Fácil", uma landing page que promove serviços de Elo boosting para o jogo League of Legends.

O projeto é composto por uma página principal *index.html* que detalha os serviços e uma página de vendas *vendas.html* com um formulário para solicitação de serviço.

# Funcionalidades Principais

## Página Principal *index.html*:

* Apresentação dos serviços (Elo Job, Duo Boost, MD5, Vitórias).

* Carrossel de avaliações de clientes.

* Seções de vantagens e garantias do serviço.

* Plano de fundo com vídeo em loop.

## Página de Vendas (vendas.html):

* Formulário de solicitação de serviço onde o usuário pode selecionar o serviço, seu elo atual e preencher dados de contato.

## Carrossel de Feedback (scripts2.js):

* Um carrossel de depoimentos de clientes totalmente funcional, controlado por JavaScript, que permite ao usuário navegar pelas avaliações usando os botões "anterior" e "próximo".

## Design Responsivo (styles.css):

* O site utiliza CSS Grid e Flexbox para organizar o conteúdo.

* Inclui media queries para adaptar o layout a dispositivos móveis (telas com largura de até 750px).

# Tecnologias Utilizadas
* HTML5: Estrutura semântica das páginas.

* CSS3: Estilização, layout (Grid/Flexbox) e responsividade.

* JavaScript (ES6+): Manipulação do DOM para a funcionalidade do carrossel de avaliações.

# Como Executar
Basta abrir o arquivo index.html em qualquer navegador web.
